Este plugin te dejará poner un versiculo en tu sitio web a traves de un shortcode. 
No permite una promesa al día, sino que al refrescar te dará otra promesa, la idea es que las personas
puedan ver una promesa diferente cada que entren al sitio. 

Caracteristicas:
Este plugin instala en la BD 50 promesas biblicas en una tabla llamada "promesa_al_dia".
Todas las promesas biblicas estan en la versión RV60
Tienes ideas o comentarios para hacer de este plugin algo más util, compartenosla al consejosdecomputo@gmail.com
